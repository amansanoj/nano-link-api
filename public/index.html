NanoLink API

A blazing-fast URL shortener and analytics engine.

Endpoints

POST /api/shorten

Converts a long URL into a random 6-character short code, or a specific custom alias if provided.

Request Body 
{
  "url": "https://hackclub.com",
  "customCode": "hack",
  "maxClicks": 3,
  "expiresAt": "2026-05-07T08:06:16.758Z"
}
GET /:code

Instantly redirects to the original destination and silently logs the click analytics.

GET /api/qr/:code

Dynamically generates and returns a scannable, high-res PNG of a QR code pointing to your short link.

GET /api/stats/:code

Retrieves live click analytics and timestamps for a specific short link.

GET /api/links

Returns a JSON array of all active short codes stored in the SQLite database.
